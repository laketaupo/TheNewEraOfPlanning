Calibrating is a **one-to-many BOM process**: a single input (Item D) fans out into three co-products — Items E, F, and G — each representing seeds of a different size grade. This branching structure is native to the o9 graph model, where a process can output any number of items. The Calibrator machine runs at 0.5 hours per batch.
