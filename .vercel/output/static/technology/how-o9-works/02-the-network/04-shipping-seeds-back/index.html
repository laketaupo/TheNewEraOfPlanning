Shipping Seeds Back

Item C is transported from the Grower back to the Central Warehouse via a BOD lane.

Supply Chain Network Graph

Item BOM BOD Resource Resource consumption
Drag nodes to reposition

Item C travels back from the Grower to the Central Warehouse through another BOD lane. The network has now crossed location boundaries twice — outbound with raw inputs, inbound with harvested seed. This return leg is modelled as a separate Ship process so its transit time and costs can be planned independently.
