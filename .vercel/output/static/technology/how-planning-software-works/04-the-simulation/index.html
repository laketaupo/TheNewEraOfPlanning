Chapter 6

The Simulation

Run what-if scenarios, stress-test your plan, and see how the supply chain responds to change.

  1. 01

    What Is a Workflow Simulation?

    A workflow simulation in Planning software is an end-to-end what-if run that lets you change assumptions, recalculate the plan, and compare the result against the baseline — without touching the live plan.

    4 min
  2. 02

    Setting Up a Simulation

    Before running a simulation in Planning software, you need to define what you are testing, create the scenario, and make the right input changes. Preparation determines whether the output is useful.

    4 min
  3. 03

    Running the Simulation

    Triggering the Planning software planning engine recalculates the supply plan end-to-end. Understanding what happens during the run helps you interpret the output correctly.

    3 min
  4. 04

    Reading Simulation Results

    The value of a simulation is in how you read and compare its output. Knowing where to look and what to compare helps you extract a clear answer from the recalculated plan.

    4 min
  5. 05

    Promoting a Simulation to Baseline

    Promotion is the step where a scenario becomes the new plan. It is the most consequential action in the simulation workflow and requires the right governance before it is taken.

    3 min
