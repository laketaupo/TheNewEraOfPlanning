Chapter 3

Data Types in Planning

Master data, transactional data, and planning parameters are three distinct data categories. Understanding the difference — and where each lives — is essential for data quality management.

  1. 01

    Master Data

    Master data is the stable, reference information that describes the objects in the planning model — items, locations, customers, suppliers. It changes slowly but when it's wrong, everything built on it is wrong.

    4 min
  2. 02

    Transactional Data

    Transactional data records what has happened: inventory movements, order receipts, production completions. It is the evidence of execution — and the foundation for keeping the plan grounded in reality.

    4 min
  3. 03

    Planning Parameters

    Planning parameters are the settings that control how the planning engine behaves for each item: lead times, safety stock, minimum order quantities, lot sizes. Getting them right is as important as having good data.

    5 min
