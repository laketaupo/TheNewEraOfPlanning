The fastest route: Tell Me (search)

The single most efficient navigation tool in ERP is Tell Me — the search box accessible via the magnifying glass icon in the top bar, or by pressing Alt + Q. Type any page name, action, or report title and ERP jumps directly to it.

Instead of navigating through Departments → Manufacturing → Planning → Purchase Orders, type "Purchase Orders" and you are there in two keystrokes. For planners who work with a consistent set of pages daily, Tell Me eliminates most menu navigation.

Bookmarks and the role centre

Pages you use frequently can be bookmarked to your role centre. Click the bookmark icon on any page to add it to your navigation bar. After a few days of use, your role centre navigation bar should reflect the 5–8 pages you open every day.

Filters and views

List pages support powerful filtering:

  • Column filters — click any column header to filter the list by that field's value
  • Filter pane — the sidebar filter panel allows multi-field filtering (e.g., show all purchase orders for supplier X with expected receipt date in the next 14 days)
  • Saved views — filter configurations can be saved as named views and recalled instantly. Creating a "Due this week" view on the purchase order list, for example, saves time compared to reapplying the same date filter each session.

Drill-down navigation

Most values in ERP lists are clickable — they open the related record. The value in the "Vendor No." column on a purchase order opens the vendor card. The quantity in the "On-Hand" column opens the item ledger entries. This drill-down navigation allows planners to trace from a high-level view to the underlying transactions without leaving the current context.

Keyboard shortcuts

ERP supports a wide set of keyboard shortcuts that speed up common operations:

Action Shortcut
Open Tell Me search Alt + Q
Open filter pane Shift + F3
Apply/remove column filter Alt + F3
Save current record Ctrl + S
New record Ctrl + N
Delete record Ctrl + Del
Expand/collapse FactBox Ctrl + F2

Learning the shortcuts for the actions you perform most often reduces session time significantly.
