The navigation structure

Planning software is organised in a layered hierarchy:

Application → Module → Workspace → View → Tab

  • The application is the top-level environment — the Planning software instance for your organisation
  • Modules group related planning functions (e.g., Demand Planning, Supply Planning, S&OP)
  • Workspaces are the primary working areas within a module — where planners spend most of their time
  • Views are specific arrangements of data within a workspace (e.g., a tabular view, a chart view, a network view)
  • Tabs within a workspace allow navigation between related datasets without leaving the current context

The left navigation panel

The left side of the Planning software screen is the primary navigation panel. It shows:

  • The module you are currently in
  • The workspaces available in that module
  • Quick links to recent views or pinned items

Clicking a workspace name opens the default view for that workspace. Most workspaces have multiple views — switch between them using the tabs or view selector within the workspace.

The top bar

The top navigation bar shows:

  • The current module and workspace name (breadcrumb)
  • The active scenario or plan version — this is critical to check before making any changes or reading results, as you may be in a scenario rather than the baseline without realising it
  • The planning horizon and currency filter (if configured)
  • User and settings access

Search and jump navigation

Planning software provides a global search that lets you jump directly to a product, location, customer, or order by name or code. This is the fastest way to navigate to a specific item in a large planning model — far faster than drilling through the hierarchy manually.

Breadcrumbs and back navigation

As you drill into specific items (a product, a customer, a supply node), Planning software maintains a breadcrumb trail at the top of the view. Use this to navigate back to a higher level without losing the filters or selections you have applied to the current view.

Keyboard shortcuts

Planning software supports keyboard shortcuts for common actions — refresh, expand/collapse rows, switch tabs. These are documented in the Planning software user guide and are worth learning for any action you perform many times per session.
